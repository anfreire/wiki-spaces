# references

## What this space is

Self-contained documentation for AI agents driving wiki-spaces operations. When a skill's procedure references one of these docs (e.g., "for mounting playbooks see `references/MOUNT.md`"), the agent fetches and follows it.

These docs are also useful for humans setting up wiki-spaces — but the primary audience is the agent helping the user.

## Items

- [SETUP.md](SETUP.md) — agent-driven setup briefing (paste-target for the README's installation prompt)
- [EXAMPLES.md](EXAMPLES.md) — canonical topology examples for personal and shared wiki shapes
- [MOUNT.md](MOUNT.md) — how to mount an external wiki as a space (clone / submodule / symlink, parent's `## Spaces` update, trust scope reminder)
- [HARNESS_INTEGRATION.md](HARNESS_INTEGRATION.md) — manual integration snippets for harnesses without skill installation (Cursor, Windsurf, GitHub Copilot, Aider)
